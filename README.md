This is my first backend project using C# and ASP.NET Core Web API. It's designed as the backend service for an ecommerce application, handling all server-side operations such as product management, user accounts, and order processing.

Features

Product Management: Add, update, delete, and list products with categories and pricing.

User Authentication: Secure login and registration for customers and admins.

Order Handling: Create and track orders with status updates.

Reviews & Ratings: Users can leave ratings and reviews for products.

Category Management: Organize products into categories like Appetizers, Main Course, Desserts, etc.

CRUD Operations: Fully functional Create, Read, Update, Delete endpoints for all major entities.

RESTful API Design: Follows standard REST conventions for easy integration with frontend apps.

Data Persistence: Uses a database (can specify MongoDB, SQL Server, etc.) to store all data efficiently.

Tech Stack

Backend: C#, ASP.NET Core Web API

Database: MongoDB

Authentication: JWT (JSON Web Tokens) for secure API access

Tools: Visual Studio / VS Code, Postman for API testing

Why I Built This

I wanted to get hands-on experience building a backend from scratch in C#. This project taught me:
